# 官网项目开发计划（plan）

## 使用说明

- 本文档放置于 skills 目录，作为每次开发前的必读计划基线
- 每次新增需求或完成大功能后，需同步更新“任务状态”与“里程碑记录”
- 已完成事项使用 `[x]`，进行中或未开始使用 `[ ]`

## 总体路线

- 阶段一（V1）：仅前端静态官网，多端适配，部署 Cloudflare
- 阶段二（V2）：接入后端 NestJS 与后台管理端，建立动态内容与线索闭环
